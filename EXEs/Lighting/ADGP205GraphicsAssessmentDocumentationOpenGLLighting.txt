Assessment ADGP 205 - Graphics


Graphics Assessment Documentation for “OpenGL Lighting”


Purpose:
Introduce the steps needed to create our own lighting models that will simulate light onto our rendered objects.


Learning Outcomes:
Learn how to setup and use specific elements from the “LightingSphereApplication” file in the shader files.
Learn how to calculate Ambient, Diffuse, and Specular lighting.
Learn how to calculate and use the reflection formula (not the built in reflection function).
Learn how to calculate Hemisphere lighting.
